## 产品经理

### 你需要做什么

- 结合公司战略及产品规划，负责公司产品的需求调研及分析、产品规划、原型设计等工作；
- 通过用户调研、数据挖掘等手段，制定并不断迭代优化产品运营策略，满足用户不同场景下的核心诉求；
- 搭建运营指标体系，发现并解决产品问题，深度参与产品设计、性能提升、程序修复等重要环节；
- 关注、分析国内外行业动态，精准把握行业趋势，协助提高产品行业地位和竞争力；
- 协调相关合作团队，管理项目进度，推进项目如期上线。

### 我们对你的期望

- 本科及以上学历，2年以上产品运营经验，互联网行业尤佳；
- 熟悉产品运营策略，热衷于全方位提升产品性能和体验；
- 重视数据驱动和结果导向，掌握或善于学习基础的数据分析工具和技巧，具有较强的数据和行业分析能力；
- 逻辑思维清晰，通过分析事物本质，精准定位问题并解决；
- 具备优秀的沟通协作能力，有产品owner意识和创新意识，自驱力强；
- 熟练掌握原型设计工具、流程图及脑图工具等（如Axure、XMind等）
- 良好的沟通表达能力和团队协作能力，与技术和研发团队有良好的合作经验；

### 加分项

- 有开发经验，或有较强的交互能力，或有较强的数据理解力

## DevOps 工程师

### 你需要做什么

- 深入了解业务，参与技术评审，对线上运维环境进行治理。
- 提升软件开发效率和质量，负责相关的方法/流程/工具研发。
- 负责 Kubernetes 及 Docker 相关技术在业务领域的方案落地、优化、开发工作，包括但不限于容器日志、容器监控、容器编排管理等；
- 负责DevOps新技术研究储备、测试验证、计划设计、开发落地及投产验证工作； 
- 聚焦于整个软件开发/交付流程，以实现全流程自动化和智能化。
- 负责各业务应用的部署、配置、运维、监控 、故障排查。
- 评估系统的性能参数，并进行深度的性能调优。

### 我们对你的期望

- 计算机、软件相关专业，本科及以上学历，1年以上相关工作经验。
- 熟悉 AWS 、阿里云、腾讯云等公有云的系统运维工作；
- 熟悉 Kubernetes 及周边生态系统，有集群开发及运维经验；
- 掌握 Python, Go, JavaScript 等一种及以上前后端语言，全栈开发经验者优先。
- 软件工程相关专业基础知识扎实，熟悉常用的提升软件质量和软件开发效率的方法和工具。
- 拥有项目管理及协作、代码托管、CI/CD、DevOps、运维、监控、大数据处理等工具开发经验者优先。
- 良好的沟通表达能力和团队协作能力，与技术和研发团队有良好的合作经验；

### 加分项

- 有强烈的责任感、能够承受较强的工作压力；
- 学习能力强、有良好的时间观念和自我管理能力；
- 熟悉企业私有云的系统运维工作，有能力解决线上IT问题。

## 高级开发工程师

### 你需要做什么

- 参与产品需求分析，研究产品技术细节；
- 参与数据库设计、缓存设计、数学建模；
- 负责业务系统的模块设计和开发；
- 改进现有服务的质量和可用率；
- 采用新技术、新的设计模式优化服务质量；

### 我们对你的期望

- 较强的组件封装能力
- 具备两个以上编程语言或框架（Typescript, Node.js, React, Koa, NestJS, Express.js）的实际项目开发经验
- 具备前后端工程化开发的能力
- 熟练使用 Git，熟悉 GitFlow
- 熟练掌握 Docker 技术
- 熟悉 RESTful API
- 对新技术有强烈的好奇心
- 可以阅读英文文档，在英语开发社区（例如 GitHub Issues，Stack Overflow）中寻找答案

#### 前端方向

- 2 年以上 React + Typescript 开发经验
- 精通 HTML5/CSS/JS
- 熟练掌握 HTTP、TCP/IP、Websocket 协议
- 熟练掌握前端工程化工具如 Webpack
- 熟练掌握前端项目性能优化的常见手段
- 了解最新的 HTML、CSS、ECMAScript 规范

> 我们的产品不需要支持 Internet Explorer。

#### 后端方向

- 2 年以上 Typescript + Node.js 开发经验
- 熟练掌握 MySQL 或 PostgreSQL 数据库设计、应用与优化，熟悉 Redis

### 加分项

- 熟练在 macOS / Linux 上开发
- Github 有项目超过 50 star，或给大型项目贡献过代码
- 了解 GraphQL
- 了解 Serverless

### 待遇

- 薪资面谈
- 标配一台 Macbook Pro （2020 款或更新） 和一台 2K 显示器
- 上午 10 点 至 晚上 6 点 30 的工作时间，节假日+双休
- 休闲小食，无酒精饮品
